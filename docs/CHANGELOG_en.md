## [1.1.0] - 2026-03-23

### Added
- **J-20 Mighty Dragon Stealth Fighter Pixel Art** — Showcasing AI's ability to create complex pixel art via natural language
  - Top view (48×48): Complete delta-wing aerodynamic layout with canards, main wings, and vertical stabilizers
  - Side view (48×32): Detailed fuselage with cockpit, air intakes, and red star insignia
  - Afterburner animation (56×32, 6 frames @ 8fps): Flickering engine exhaust flame effect
  - Sprite sheet (3×2 layout): All animation frames exported as a single image
- New J-20 fighter showcase section on the official website Examples page
- Bilingual changelog support (English & Chinese)

### Improved
- Website now supports multi-view showcase layouts for top view / side view / animation combinations
