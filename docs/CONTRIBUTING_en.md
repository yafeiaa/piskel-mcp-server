# Contributing to Piskel MCP Server

[English](./CONTRIBUTING_en.md) | [简体中文](./CONTRIBUTING_zh.md)

## Development Setup

```bash
# Clone the repository
git clone https://github.com/yafeiaa/piskel-mcp-server.git
cd piskel-mcp-server

# Install dependencies
npm install

# Build
npm run build

# Run tests
npm test
```

## Code Style

- Use TypeScript
- Follow existing code conventions
- Run `npm run lint` before committing

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests and linting
5. Submit a pull request

## Reporting Issues

Please report issues on [GitHub Issues](https://github.com/yafeiaa/piskel-mcp-server/issues).
