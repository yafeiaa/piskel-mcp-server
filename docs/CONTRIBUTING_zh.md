# 贡献指南

[English](./CONTRIBUTING_en.md) | [简体中文](./CONTRIBUTING_zh.md)

## 开发环境搭建

```bash
# 克隆仓库
git clone https://github.com/yafeiaa/piskel-mcp-server.git
cd piskel-mcp-server

# 安装依赖
npm install

# 编译
npm run build

# 运行测试
npm test
```

## 代码风格

- 使用 TypeScript
- 遵循现有代码规范
- 提交前运行 `npm run lint`

## Pull Request

1. Fork 仓库
2. 创建功能分支
3. 进行修改
4. 运行测试和 lint
5. 提交 Pull Request

## 报告问题

请在 [GitHub Issues](https://github.com/yafeiaa/piskel-mcp-server/issues) 上报告问题。
