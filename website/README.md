# Piskel MCP Server Website

This is the official website for Piskel MCP Server.

## Development

Simply open `index.html` in a browser to view the website locally.

## Deployment

This website is designed for GitHub Pages deployment. The website uses raw GitHub URLs for images, so it will work immediately when deployed.

### Option 1: Use as root

Push the contents of this folder to the `gh-pages` branch.

### Option 2: Use /docs folder

Copy the contents of this folder to `docs/` at the repository root and push to the main branch.

## Features

- Bilingual support (English/Chinese)
- Responsive design
- Pixel-art inspired aesthetics
- Live preview in hero section
- Example gallery showcasing pixel art created with Piskel MCP Server
- Quick start guide
- Complete MCP tools reference
