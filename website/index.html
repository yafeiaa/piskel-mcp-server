强大功能

创作专业像素艺术所需的一切工具

绘图工具

像素、线条、矩形、圆形和洪水填充。完整的绘图工具，满足任何像素艺术项目需求。

图层管理

使用多个图层组织你的作品。添加、删除和重新排序图层，创建复杂构图。

动画制作

创建逐帧动画。复制、添加和删除帧，实现流畅的精灵动画。

导出选项

导出为 PNG、GIF 和精灵图。非常适合游戏开发和 Web 项目。

AI 集成

由模型上下文协议驱动。连接 AI 助手，使用自然语言创作艺术。

开发者友好

通过 MCP 提供简单的 API。易于集成到你自己的工作流程或扩展自定义需求。

快速开始

几分钟内启动并运行

1

安装

git clone https://github.com/yafeiaa/piskel-mcp-server.git
cd piskel-mcp-server
npm install
2

构建

npm run build
3

配置 AI

将以下配置添加到你的 Agent 设置中：

{
  "mcpServers": {
    "piskel": {
      "command": "node",
      "args": ["/path/to/piskel-mcp-server/dist/index.js"]
    }
  }
}
4

创作艺术

开始使用自然语言创建像素艺术：

"创建一个 32x32 的红色爱心精灵"
"在中心画一个蓝色圆形"
"导出为动画 GIF"
